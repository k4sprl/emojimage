## Emoji To Image
You can convert emojis to images and CDN links by sending the emoji/s to the bot via DM or by typing `/e2img :emojiname:` (up to 5 at once) in any DM or channel that allows you to do so. And lastly you can also right-click (*or hold on mobile*) any message which contains emoji/s and select `Apps>Emojimage`.

## Sticker To Image
You can convert stickers to images and CDN links by sending the sticker to the bot via DM or by typing `/s2img` in any DM or channel that allows it. And lastly you can also right-click (*or hold on mobile*) any sent sticker and select `Apps>Emojimage`.

<img src="https://github.com/k4sprl/emojimage/blob/d698c372dcc43e5966f1e98e1360e9f4f7f3af7a/stickerShowcase.gif" width="500">

## Commands
|Command|Explaination|
|:---------:|:---------|
|**/about**|Shows about text *(only visible to you)*|
|**/e2img**|Extract images out of emojis|
|**/help**|Shows help message|
|**/invite**|Add bot to your server or apps|
|**/ping**|Check bot latency|
|**/privacy**|Link to Privacy Policy|
|**/s2img**|Extract images out of stickers|
|**/share**|Share Emojimage with others *(sends a message with info and invite button)*
|**/source**|Link to source code / GitHub
|**/tos**|Link to Terms of Service|
|**/website**|Link to Website|

Tip: You can also acces commands via the Apps button on the bottom right of your message bar.
